# Storytelling

Crafts compelling narratives using proven story frameworks and techniques. Guides structured narrative development for any purpose—brand narratives, user stories, change communications, or creative fiction.

## Story Framework Library

### Transformation Narratives

| Framework | Best For |
|-----------|----------|
| **Hero's Journey** | Classic transformation with adventure and return |
| **Pixar Story Spine** | Emotional structure building to resolution |
| **Customer Journey Story** | Before/after transformation |
| **Challenge-Overcome Arc** | Obstacle-to-victory structure |

### Strategic Narratives

| Framework | Best For |
|-----------|----------|
| **Brand Story** | Values, mission, unique positioning |
| **Pitch Narrative** | Persuasive problem-to-solution |
| **Vision Narrative** | Future-focused aspirational story |
| **Origin Story** | Foundational narrative |

### Specialized Narratives

| Framework | Best For |
|-----------|----------|
| **Data Storytelling** | Transform insights into narrative |
| **Emotional Hooks** | Powerful opening and touchpoints |
| **Three-Act Structure** | Classic dramatic structure |
| **Story Brand** | Clear customer transformation |

## Storytelling Principles

- **Conflict is essential** - Every great story has tension
- **Show, don't tell** - Use vivid, concrete details
- **Change is essential** - What transforms?
- **Emotion drives memory** - Find the feeling
- **Authenticity resonates** - Stay true to core truth

## Usage

1. **Define purpose** - Why is this story being told?
2. **Know audience** - Who will experience it?
3. **Select framework** - Match framework to purpose
4. **Gather elements** - Characters, conflict, change
5. **Build arc** - Structure with tension and resolution
6. **Adapt to platform** - Tailor to medium

## Framework Selection Guide

| Purpose | Recommended Framework |
|---------|----------------------|
| Marketing/brand | Brand Story, Hero's Journey |
| Product pitch | Pitch Narrative, Challenge-Overcome |
| User case study | Customer Journey, Three-Act |
| Vision/strategy | Vision Narrative, Story Brand |
